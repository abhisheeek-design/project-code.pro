# CodeQuest Flutter

Flutter scaffold for the CodeQuest mobile app.

## Included

- `MaterialApp` dark theme using `Color(0xFF0D0D0D)`
- Pixel-style level select screen
- HTML, Java, and Python tracks
- Bordered cards, progress bars, locked levels, and `START` buttons

## Next steps

1. Add `assets/fonts/PressStart2P-Regular.ttf`, or remove the custom font entry from `pubspec.yaml`.
2. Run `flutter pub get`.
3. Run `flutter run`.

Judge0, accounts, real lesson persistence, and certificate generation are intended backend integrations beyond this static scaffold.
